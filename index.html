📖 Manuale d'uso — ForgeKit

Due strumenti in uno · tutto offline · nessun dato inviato su internet

✏️ Tab 1 — Rinomina File

  • Carica file: trascina o clicca sul riquadro
  • Attiva regole a sinistra: prefisso, suffisso, numerazione, maiuscole, cerca&sostituisci, pulizia
  • Anteprima live: vedi subito il nuovo nome
  • Applica: 1 file → download diretto · più file → ZIP automatico

🔄 Tab 2 — Converti File

  • Scegli la conversione dal menu a sinistra
  • Carica i file (anche 20+ insieme)
  • Regola qualità e colore sfondo per le immagini
  • Clicca Converti & Scarica → ricevi un ZIP con tutti i file convertiti
  • Conversioni disponibili: JPG/PNG/WebP/SVG, CSV↔JSON, TXT→MD, Base64, ZIP

❓ FAQ

I file vengono inviati su internet? No, tutto avviene nel browser.
Funziona offline? Sì, 100% offline dopo il primo caricamento.
I file originali vengono modificati? No, vengono create nuove copie.
Quanti file posso convertire insieme? Nessun limite, ma molti file grandi richiedono più tempo.
WebP è supportato da tutti i browser? Sì, da Chrome, Edge, Firefox e Safari moderni.
